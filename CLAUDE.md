# machax (GrowthX AI Weekender)

Sprint project for the GrowthX AI Weekender hackathon.

## key files

- `./handbook/` — full sprint handbook (ideas, rubric, setup, build pipeline)
- `./weekender.md` — participant's live state (track, idea, stage, URL, metrics, daily log)

## constraints

- Writing style: lowercase headings, direct, no corporate tone
- Do NOT fill in participant's thinking — ask, don't assume
- `weekender.md` is participant-owned; only update "live state" and "daily log" sections with confirmed facts
- Re-read `weekender.md` before responding to: "coach me", "check in", "where are we", "where am I", "what's next"

## commands

Update handbook:
```
curl -fsSL https://raw.githubusercontent.com/GrowthX-Club/ai-weekender-handbook/main/install.sh | bash
```
